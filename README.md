# APB-Interfaced-SPI-Master-IP-Core-RTL-Design-and-Verification  

## Overview  

This repository contains the **Register Transfer Level (RTL) Design** and **UVM-based Verification** of an **SPI Master IP Core** integrated with the **Advanced Peripheral Bus (APB)** interface.  
The **SPI Master IP Core** is designed to provide configurable, high-performance communication with SPI slave devices through an APB-compliant interface, making it easy to integrate into AMBA-based SoCs.
